## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Purujeet-git/Internship-Test.git
```

Move into the project folder:

```bash
cd Internship-Test
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```
